## What it does

Feed it a decision — pricing, hiring, market entry, pivot, partnership — and it returns a structured analysis designed to surface what you're missing.

### Output for every decision:

| Section | What you get |
|---------|-------------|
| **Frame** | What's actually being decided (vs. what you think) |
| **Options** | All viable paths, including the ones you haven't considered |
| **Stress test** | Each option tested against your real constraints |
| **Risks** | Hidden risks and second-order consequences |
| **Recommendation** | A clear path with confidence level and key assumptions |

---

## Built for

Founders making high-stakes calls without a board, advisor, or experienced operator in the room.

> The best decisions aren't the ones with the most data. They're the ones where you've correctly identified what you're actually deciding.
